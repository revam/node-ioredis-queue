# ioredis-queue

Simple redis queue system

## Install

```sh
npm install --save https://github.com/revam/node-ioredis-queue.git
```

## What is this?

This is a simple queue system using redis for message transport.

## Usage

```js
import { RedisQueue } from "ioredis-queue";
```
